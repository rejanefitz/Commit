Commit Git
Lorsque vous modifiez votre repository, vous devez demander à Git d'enregistrer vos modifications en faisant un git commit.

Pour faire un commit git commit -m
 L'option-m vous permet de lui envoyer un message décrivant les modifications effectuées.

Evidemment , avant de commit, il faudras avoir un repository.
